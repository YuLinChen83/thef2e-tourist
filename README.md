# WebRTC Practice
From 1-to-1 to many-to-many video chat webpage built with WebRTC, React and Firestore.

## Available Commands
* `npm install`
* `npm run dev`

## Built With
* React 7.0.0
* Typescript 4.3.2
* Firebase 9.0.2
* Vite 2.5.4
